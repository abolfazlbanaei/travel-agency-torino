### Work Locally 💻

1.  Download or clone the repository

```bash
git clone https://github.com/abolfazlbanaei/travel-agency-torino.git
```

2.  Install dependencies

```bash
npm install
```

3.  start the local server

```bash
npm run dev
# http://localhost:3000
```
